# 「施展体系」我的人生灯塔

## 什么是体系？

阶段1：碎片化（点）
```
┌─────────────────────────────────────┐
│  ●    ●        ●    ●        ●      │
│      ●            ●    ●            │
│  ●        ●    ●        ●    ●      │
│      ●    ●            ●            │
│  ●        ●    ●    ●        ●      │
└─────────────────────────────────────┘
```
无规则的小点，零散、独立、随机分布

做零碎的事情 像一个个小点 无规则无规律

是不是大家常见的工作状态，每天都很忙，感觉被零碎的事情充斥，同时感觉事情根本做不完

比如我能完成上级或他人安排的某些具体工作，可能需要他人主动辅导


阶段2：线性化（点到线）
```
┌─────────────────────────────────────┐
│  ●────●────●       ●    ●        ●  │
│                                     │
│  ●        ●    ●          ●────●    │
│                                     │
│  ●────●────●    ●    ●    ●   ●   ● │
└─────────────────────────────────────┘
```
小点开始连接成线，形成简单的线性关系

从完成一件件特定的事情，到能逐步负责一片小领域

比如我能完全自己主导完成部分工作，不再强依赖上级或他人的具体安排或辅导


阶段3：网络化（线到多线）
```
┌─────────────────────────────────────┐
│  ●──●──●    ●    ●───────●     ●    │
│  │  │  │                            │
│  ●──●──●────●      ●     ●────●     │
│  │  │  │                            │
│  ●──●──●    ●──●──●    ●──●──●      │
└─────────────────────────────────────┘
```
多条线逐步闭环，形成一个局部小面

能逐步做好几类事情，逐步在局部形成小的的知识体系，开始独立负责某个领域

独立负责领域A 不一定能很好负责领域B

沉淀能负责领域A的知识体系

沉淀能负责领域B的知识体系

或者...继续沉淀

演绎出 一套可以适用于任意领域的知识体系

从解决 A 问题、从解决 B 问题 ...等等 演绎到可以解决任何问题

体系 是 量变到质变的结果 是把一个个零碎的点 串成线 构成面的过程 长期且辛苦

阶段4：体系化（线到面）
```
┌─────────────────────────────────────┐
│   ●────●────●────●────●────●────●   │
│   │    │    │    │    │    │    │   │
│   ●────●────●────●────●────●────●   │
│   │    │    │    │    │    │    │   │
│   ●────●────●────●────●────●────●   │
│        完整的知识/技能体系             │
└─────────────────────────────────────┘
```
多条线相互连接，形成完整的面状结构

融会贯通能体系化的做事情 小点构成好几条线 几条线构成面

## 什么是「施展体系」？

为此，毕业这十年我构建出了「施展体系」。一个属于我个人，可以**适应任何环境**和**做任何事情**的个人体系化解决方案。

目的： 支持我长期坚持正确的事情，推动我不断接近自己的人生目标。

- 1～5年：技术专家路线（技术轮），完成✅
- 5～10年：较为成熟的管理者路线（管理轮），完成✅
- 10～20年：能操盘年营收十万级别生意（业务轮），进行中🚗

目标：

- 20岁～50岁：成为懂技术/懂经营/懂管理的复合型技术人才
